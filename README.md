# AI-Phishing-Detector
An phishinh detection web application built with flask. This project simulates how cybersecuirty tools analyze suspicious emails by identifying potential threats.
---

## Features

- AI-inspired phishing analysis
- Critical Threat detection
- Suspicious Activity detection
- Safe email classification
- Interactive cyber-themed interface
-  Modern glowing dashboard UI
-  AI scanning simulation

---

## Programs used

- Python
- Flask
- HTML
- CSS
- JavaScript

---

## How It Works

Users paste the contents of an email or messsage that looks suspicious into the application.

The system analyzes the text for common phishing indicators such as:

- Urgent requests
- Credential verification
- Password requests
- Banking keywords
- Suspicious language

The application then classifies the email into one of three categories:

- Critical Threat
- Suspicious Activity
- Safe
